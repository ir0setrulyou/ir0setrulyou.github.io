记录GitHub pages托管个人博客过程

由于个人的服务器上托管了其它的服务，单独拿一个服务器来托管博客网站过于浪费，所以想到在GitHub pages托管博客网站，看了很多个博客框架的介绍，最终选择Hexo作为框架进行构建，选择fluid作为默认主题。

GitHub

打开GitHub网站，点击sign up注册成功后会跳转到主界面，创建新的仓库，仓库名称必须是用户名.github.io选择public，下面选项根据个人需求选择。完成后点击Create repository，以上就完成了个人仓库的一个创建。转为新建的仓库界面，点击settings找到pages就可以看到自己的用户名.github.io显示了。此时可以点击访问，看看效果如何。

安装Hexo

在安装Hexo框架之前，我们需要安装node.js，打开node.js网站，根据个人使用的系统下载相应的安装包，按步骤安装完成后按下win键搜索cmd在命令行中输入node，如果看到以下输出，说明node.js安装成功。

下一步安装Hexo,打开cmd或者power shell，输入npm install hexo-cli -g根据命令操作安装完成后，在命令行中输入hexo,返回以下输出，说明Hexo安装成功。

接下来在命令行打开自己想要创建博客的文件夹，输入hexo init blog，blog可以换成自己想要的名字，最好是英文，接着输入cd blog，转到博客目录后，输入npm install接下来会安装Hexo所需的组件，安装完成后输入hexo server，输出以下界面，Hexo安装成功，按住ctrl单击http://localhost:4000，成功打开Hexo。

安装fluid主题
